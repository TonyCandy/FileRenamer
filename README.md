有以下功能：
1、可以批量重命名指定路径下的子文件夹
2、可以批量重命名指定路径下的所有文件
使用pyinstaller --onefile --windowed main.py打包成exe文件
Python 3.13
安装 PyQt5 和 pandas
pip install PyQt5 pandas
安装 PyInstaller
pip install pyinstaller
在项目文件夹中执行：pyinstaller --onefile --windowed main.py
